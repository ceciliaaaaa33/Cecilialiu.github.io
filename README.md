hiA modern, responsive personal portfolio website featuring interactive data visualizations and creative SVG art.
This is a personal portfolio website created as part of an A2 assignment. The site showcases personal information, skills, experience, and creative data visualizations built using HTML, CSS, and JavaScript.

--Features--
Main Website (index.html)

Responsive Navigation: Fixed navigation bar with smooth scrolling
Header Section: Eye-catching personal introduction
About Me Section: Personal profile and background
Education Timeline: Visual timeline of educational background
Skills Grid: Technical skills categorized by type with interactive labels
Experience Cards: Professional experience and projects
Contact Form: Functional contact form with real-time validation

--Visualizations Page (visualizations.html)--

Temperature Bar Chart: Interactive SVG bar chart displaying monthly temperature trends

Animated bars with gradient fills
Hover toggle displays precise temperature values
Responsive design

--Technologies Used--

HTML5: Semantic markup and structure
CSS3: Modern styling using custom properties, gradients, and animations
JavaScript: Interactive functionality
SVG: Scalable vector graphics visualization

--Key Features--

Smooth Scrolling: All anchor links enable seamless scrolling to corresponding sections.
Responsive Layout: Adapts to tablet and desktop screens.
Animations: Fade-in effects, hover states, and transition effects.
Interactive Visualizations: SVG-based charts with hover effects.

--Usage Instructions--

Clone or download the repository
Open index.html in a web browser
Browse sections using the navigation menu
Click “Visualizations” to view data visualization pages
